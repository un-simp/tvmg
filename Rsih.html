RSIH

Return to the Spectrum of Intergalactic Happiness

About

Get ready for the second ever community-made live event in Beat Saber, using the Synapse mod! RSIH is a 7 map modchart pack, featuring tracks from Zeds Dead packed with intense visuals and full Vivify effects to make the experience completely unforgettable.

Made by TVMG — The Vivify Mapping Group.

Setup

To play the maps you'll need the following mods installed in Beat Saber:

  • Vivify — custom asset bundles, shaders, and visual effects
  • Noodle Extensions — note and environment animations
  • Chroma — custom lighting and colours
  • Synapse — live event multiplayer sync (required for live events)
